## Browser upload fallback

If you are creating the repo manually in GitHub:

1. Create a new empty public repo named `axiomos-lite`
2. Upload the contents of this folder as the repo root
3. Keep `index.html`, `manifest.json`, `sw.js`, `css/`, `icons/`, `js/`, `tools/`, and the three `.cmd` files together
4. Verify the README renders the screenshots from `assets/preview/`
5. Add the Gumroad full link in the repo About section
